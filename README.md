* aws cdk stack infrastructure
* deployed lambda api
* react frontend

## Api
### Install Dependencies
Navigate to the `infrastructure/` directory
`npm i`

### Deploy
`cdk deploy`

### Destroy
`cdk destroy`

## Front End
### Install Dependencies
Navigate to the `website/` directory
`npm i`

### Build/Run Locally
`npm start`

### Deploy
`npm run build-and-deploy`
